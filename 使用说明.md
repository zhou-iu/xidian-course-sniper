# 选课助手

Tampermonkey 用户脚本，用于在选课页手动启动后，按指定优先级持续尝试选择教学班。

当前版本：`v2.1.1`

## 当前目标优先级

脚本会按以下顺序尝试，抢到任意一个后立即停止：

1. `[36] 胡景钊, 万波, 李荣涵, 李飞, 张晨曦`
2. `[07] 黄超`
3. `[57] 刘慧`
4. `[50] 李航, 黄伯虎, 赵佩佩, 杜军朝`

如果当前目标返回满员、容量不足、无余量等结果，会自动切到下一个候选教学班。

## 安装

1. 在 Chrome 或 Edge 安装 Tampermonkey（篡改猴）扩展。
2. 打开扩展的“添加新脚本”，删除编辑器中的示例内容。
3. 将用户脚本文件的全部内容粘贴进去并保存。
4. 刷新选课页面，确认右下角显示选课助手面板和版本号。

脚本不保存账号、密码或 Cookie，也不会处理验证码。

## 使用

1. 打开选课系统，正常登录并进入当前轮次选课页。
2. 在页面中让目标教学班出现在列表里。脚本会从当前课程列表和接口返回数据中捕获候选教学班。
3. 右下角显示“已锁定教学班”后，点击“启动”。
4. 脚本会立即开始提交，不设预定启动时间。

当前提交间隔为约 `120–220ms` 随机重试；当前目标满员后约 `60ms` 切到下一个候选。

脚本使用网站自身的请求组件提交，因此“未开始选课”“人数已满”等提示由网站原生逻辑显示；助手面板同时记录最近一次结果。

## 停止条件

脚本会在以下情况停止：

- 已在“已选课程”或网站实时数据中确认抢到目标教学班；
- 所有候选教学班都已满或未捕获到可用数据；
- 登录失效、无权限、时间冲突、学分限制等明确不可继续的错误；
- 达到 8 小时运行上限；
- 用户手动点击“停止”。

## 注意

- 开选期间不要退出登录、关闭选课标签页或让电脑休眠。
- 请只保留一个运行脚本的选课标签页，避免重复请求。
- 如果面板一直显示“未捕获”，点击“重新捕获”，然后重新搜索一次课程。
- 如果右下角版本号不是最新版，说明 Tampermonkey 中的脚本还没更新。
